## hexist ##

**Interface:** `/hexist`

**Method:** `GET`

**Parameter:** 

*  **tb** (Required)  
*  **key** (Required)  

This Interface is an proxy interface for `/hustdb/hexist`. See more details in [here](../hustdb/hustdb/hexist.md).  

**Sample:**

    curl -i -X GET "http://localhost:8082/hexist?tb=test_table&key=test_key"
