## keys ##

**Interface:** `/keys`

**Method:** `GET`

**Parameter:** 

*  **peer** (Required)  
This represents the indexes of backend servers. See more details in [peer_count](peer_count.md)
*  **offset** (Required) 
*  **size** (Required)
*  **file** (Required)
*  **start** (Optional)
*  **end** (Optional)
*  **noval** (Optional)
*  **async** (Optional)

This Interface is an proxy interface for `/hustdb/keys`. See more details in [here](../hustdb/hustdb/keys.md).
