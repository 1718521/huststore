## del ##

**Interface:** `/del`

**Method:** `GET`

**Parameter:** 

*  **key** (Required)  
*  **ver** (Optional)  

This Interface is an proxy interface for `/hustdb/del`. See more details in [here](../hustdb/hustdb/del.md).  

**Sample:**

    curl -i -X GET "http://localhost:8082/del?key=test_key"
